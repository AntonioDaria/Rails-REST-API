# RAILS 5 REST API

This simple rails project is a basic REST API bilt to perform CRUD functions. Uses postgresql.

## RUBY VERSION
```ruby 2.5.1p57
```

## INSTALL DEPENDENCIES
```bash
$ bundle install
```

## RUN MIGRATION
```bash
$ rails db:migrate
```

## RUN SERVER
```bash
$ rails s
